# Intern Daily Report Platform

This repository contains the Google Apps Script based Intern Daily Report Platform for Globsyn Business School.

## Modules
- Intern Daily Report Submission
- Manager OTP Authentication
- Manager Review Dashboard
- Approved Report PDF Export
- Google Sheet Database Integration

## Deployment
This project must be deployed through Google Apps Script Web App, not directly through GitHub Pages, because it uses:
- SpreadsheetApp
- MailApp
- CacheService
- google.script.run
